### 문제 설명

<p>666은 종말을 나타내는 수라고 한다. 따라서, 많은 블록버스터 영화에서는 666이 들어간 제목을 많이 사용한다. 영화감독 숌은 세상의 종말 이라는 시리즈 영화의 감독이다. 조지 루카스는 스타워즈를 만들 때, 스타워즈 1, 스타워즈 2, 스타워즈 3, 스타워즈 4, 스타워즈 5, 스타워즈 6과 같이 이름을 지었고, 피터 잭슨은 반지의 제왕을 만들 때, 반지의 제왕 1, 반지의 제왕 2, 반지의 제왕 3과 같이 영화 제목을 지었다. 하지만 숌은 자신이 조지 루카스와 피터 잭슨을 뛰어넘는다는 것을 보여주기 위해서 영화 제목을 좀 다르게 만들기로 했다.</p>

<p>종말의 수란 어떤 수에 6이 적어도 3개 이상 연속으로 들어가는 수를 말한다. 제일 작은 종말의 수는 666이고, 그 다음으로 큰 수는 1666, 2666, 3666, .... 이다. 따라서, 숌은 첫 번째 영화의 제목은 "세상의 종말 666", 두 번째 영화의 제목은 "세상의 종말 1666"와 같이 이름을 지을 것이다. 일반화해서 생각하면, N번째 영화의 제목은 세상의 종말 (N번째로 작은 종말의 수) 와 같다.</p>

<p>숌이 만든 N번째 영화의 제목에 들어간 수를 출력하는 프로그램을 작성하시오. 숌은 이 시리즈를 항상 차례대로 만들고, 다른 영화는 만들지 않는다.</p>

### 입력 

 <p>첫째 줄에 N이 주어진다. N은 10,000보다 작거나 같은 자연수이다.</p>

### 출력 

 <p>첫째 줄에 N번째 영화의 제목에 들어간 수를 출력한다.</p>

---

### 다른 풀이법
```python
import sys
input = sys.stdin.readline

n = int(input())

c, i = 0, 0

while c != n:
    # 디버깅
    if i == 66:
        pass

    if i % 10 != 6:
        t = i * 1000 + 666
        c += 1

    else:
        six, tmp = 1, i
        while tmp % 10 == 6:
            six *= 10
            tmp //= 10

        for j in range(six):
            t = i * 1000 + 666 // six * six + j
            c += 1
            if c == n: break

    i += 1

print(t)
```

### 메모

#### 1. 왜 `else`가 필요한가? (패턴의 변화)

`i`라는 숫자를 "접두사(Prefix, 앞부분)"라고 생각하고 뒤에 숫자를 붙여봅시다.

* **일반적인 경우 (`if` 블록)**
  * `i = 1` → 뒤에 `666`만 붙이면 됨 → `1666` (1개 생성)
  * `i = 2` → `2666` (1개 생성)
  * ...
  * `i = 5` → `5666` (1개 생성)


* **특수한 경우 (`else` 블록: `i`가 6으로 끝날 때)**
  * `i = 6`일 때, 단순히 `6666` 하나만 있는 게 아닙니다.
  * `6660`, `6661`, `6662`, ... , `6669`
  * 이렇게 **10개의 숫자**가 한꺼번에 쏟아져 나옵니다.
  * 이 구간을 하나씩 다 세기 위해 `for` 문이 필요한 것입니다.


#### 2. 코드 한 줄씩 해부하기

`i = 6`인 상황을 가정하고 코드를 따라가 보겠습니다.

**① 연속된 6의 개수 세기 (`six` 변수)**

```python
six, tmp = 1, i
while tmp % 10 == 6:  # 끝자리가 6인가?
    six *= 10         # 자릿수 확장 (1 -> 10 -> 100...)
    tmp //= 10        # 끝자리 자르기
```

* **목적:** 접두사 `i`의 끝에 6이 몇 개나 연속으로 붙어있는지 확인합니다.
* **`i=6`일 때:** `tmp`가 6이므로 `six`는 **10**이 됩니다. (범위가 0~9까지 10개라는 뜻)
* **`i=66`일 때:** 6이 두 개이므로 `six`는 **100**이 됩니다. (범위가 00~99까지 100개라는 뜻)

**② 범위만큼 숫자 생성하기 (`for` 문)**

```python
for j in range(six): # six가 10이면 0~9까지 반복
    # 여기가 가장 복잡한 수식
    t = i * 1000 + (666 // six) * six + j

```

`i = 6`, `six = 10`일 때 이 수식을 풀어보겠습니다.

1. **`i * 1000`**: `6000` (앞자리 확보)
2. **`666 // six`**: `666 // 10` = `66` (몫)
3. **`... * six`**: `66 * 10` = `660` (다시 곱해서 자릿수 맞춤)
* *왜 이렇게 복잡하게 하냐구요?* 뒤의 일의 자리를 `0`으로 비워두기 위해서입니다. (`666` -> `660`)


4. **`+ j`**: `j`는 0부터 9까지 변합니다.

**결과:**

* `j=0`: `6000 + 660 + 0` = **6660**
* `j=1`: `6000 + 660 + 1` = **6661**
* ...
* `j=9`: `6000 + 660 + 9` = **6669**

이렇게 **`666`이 포함된 숫자 10개를 한 번에 생성**해서 카운트(`c`)를 올립니다.
