# 파이썬 내장 함수 정리


## divmod(a, b)

### 개요

두 수를 나눈 **몫(quotient)** 과 **나머지(remainder)** 를 **튜플**로 동시에 반환하는 함수.

```python
divmod(a, b)  # → (a // b, a % b)
```

---

### 기본 사용법

```python
divmod(10, 3)   # → (3, 1)   → 몫: 3, 나머지: 1
divmod(7, 2)    # → (3, 1)
divmod(20, 5)   # → (4, 0)   → 나머지 0 (나누어 떨어짐)
divmod(1, 5)    # → (0, 1)   → 몫이 0인 경우
```

```python
q, r = divmod(17, 4)
print(q)  # 4
print(r)  # 1
```

---

### 음수 처리

Python의 `//` 와 `%` 는 **내림(floor) 방식**이므로, 음수가 포함되면 수학적 정의를 따름.

```python
divmod(-7, 3)   # → (-3, 2)   ← 수학적 floor division
divmod(7, -3)   # → (-3, -2)
divmod(-7, -3)  # → (2, -1)
```

> 확인 공식: `b * q + r == a` 가 항상 성립
>
> - `-3 * (-3) + 2 == 9 + 2 == 11` ← 아니고,
> - `3 * (-3) + 2 == -9 + 2 == -7` ✓

---

### 실수(float) 지원

정수뿐 아니라 `float`도 사용 가능. 반환되는 몫은 소수점이 제거된 float.

```python
divmod(5.5, 2)    # → (2.0, 1.5)
divmod(7.5, 2.5)  # → (3.0, 0.0)
divmod(-5.5, 2)   # → (-3.0, 0.5)
```

---

### `divmod` vs `//` + `%` 따로 쓰기

| 방식 | 코드 | 특징 |
| --- | --- | --- |
| `divmod` | `q, r = divmod(a, b)` | 한 번의 연산으로 몫과 나머지 동시 획득 |
| 따로 계산 | `q = a // b` `r = a % b` | 두 번의 연산 |

내부적으로 `divmod`는 단일 연산이라 **성능상 미세하게 유리**하고, 코드가 더 명확함.

---

### 실전 활용 패턴

#### 1. 시간 단위 변환

```python
total_seconds = 3661

minutes, seconds = divmod(total_seconds, 60)
hours, minutes = divmod(minutes, 60)

print(f"{hours}시간 {minutes}분 {seconds}초")  # 1시간 1분 1초
```

#### 2. 진법 변환 (10진수 → n진수)

```python
def to_base(n, base):
    digits = []
    while n > 0:
        n, remainder = divmod(n, base)
        digits.append(remainder)
    return digits[::-1]

to_base(255, 16)  # [15, 15] → 0xFF
to_base(10, 2)    # [1, 0, 1, 0] → 0b1010
```

#### 3. 배열 인덱스 → 행/열 변환 (2D 그리드)

```python
cols = 4
for i in range(12):
    row, col = divmod(i, cols)
    print(f"index {i:2d} → ({row}, {col})")
```

```text
index  0 → (0, 0)
index  1 → (0, 1)
...
index  5 → (1, 1)
...
index 11 → (2, 3)
```

---

### 타입 제약

```python
divmod("10", 3)   # TypeError: unsupported operand type(s)
divmod(10+2j, 3)  # TypeError: complex 타입은 지원 안 함
```

- 지원 타입: `int`, `float`
- 미지원 타입: `complex`, `str`, `list` 등

---

### 핵심 요약

| 항목 | 내용 |
| --- | --- |
| 반환값 | `(몫, 나머지)` 튜플 |
| 동치 | `(a // b, a % b)` |
| 음수 | floor division 기준 (수학적 정의) |
| float | 지원 O |
| complex | 지원 X |
| 주요 용도 | 시간 변환, 진법 변환, 2D 인덱싱 |
