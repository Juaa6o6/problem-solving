### 배운 내용
* 이스케이프 시퀀스(escape sequence)<br>

  - 출력 시에 특별한 제어 기능을 가진 문자열
  - 이스케이프 시퀸스에 사용되는 백슬래쉬(\)는 escape character이라고 불림
  - 백슬래쉬(\) + 특정 문자열로 이스케이프 시퀸스를 나타냄
  - 이스케이프 시퀸스 종류<br>
    + \\\ : 백슬래쉬 출력
